TFR System Setup

Connect your Supabase database — one time only

1
Go to supabase.com → Sign up free → Create a new project → Choose any region close to India.
2
In your project, go to SQL Editor → click New Query → paste and run the SQL below to create the TFR table.

Show SQL to run ▾ 
CREATE TABLE tfr_requests (
  id TEXT PRIMARY KEY,
  tool_no TEXT,
  tool_name TEXT,
  requester TEXT,
  department TEXT,
  quantity INTEGER,
  revision TEXT,
  required_date TEXT,
  priority TEXT,
  status TEXT DEFAULT 'Pending',
  planned_date TEXT,
  commit_date TEXT,
  completion_date TEXT,
  execution_type TEXT,
  remarks TEXT,
  created_at TIMESTAMPTZ DEFAULT NOW(),
  updated_at TIMESTAMPTZ DEFAULT NOW()
);

ALTER TABLE tfr_requests ENABLE ROW LEVEL SECURITY;
CREATE POLICY "Allow all" ON tfr_requests FOR ALL USING (true) WITH CHECK (true);
3
Go to Project Settings → API. Copy your Project URL and anon public key and paste below.
